# Three.js Black Hole with Shaders

A Pen created on CodePen.

Original URL: [https://codepen.io/VoXelo/pen/wBKvJxd](https://codepen.io/VoXelo/pen/wBKvJxd).

An interactive 3D black hole visualization created using Three.js and WebGL. This demo utilizes custom GLSL shaders to render the animated accretion disk and the gravitational lensing effect. Post-processing with UnrealBloomPass adds the vibrant glow.
